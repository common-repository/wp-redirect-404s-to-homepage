WordPress plugin redirect and 404 pages to the website homepage.

== Description ==

This WordPress Plugin is meant for catching any 404 page visits and redirecting them to the homepage. 

Major features in WP Redirect 404s To Homepage include:

* Reduces the amount of 404s your domain receives.
* No setup required!

== Installation ==

Upload the "wp-redirect-404-homepage" folder to your wp-content/plugins directory, then Activate it.
